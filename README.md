# NoDeskClaw

OpenClaw instance visual management system — deploy, monitor, and orchestrate AI employees through a 3D hexagonal workspace.

## Vision

NoDeskClaw transforms how teams interact with AI agents by providing:

- **One-click deployment** of OpenClaw instances on Kubernetes
- **3D hexagonal workspace** for spatial organization of AI employees
- **Gene ecosystem** for skill management and evolution
- **Real-time collaboration** between humans and AI through corridor topology

## Tech Stack

- **Backend**: Python 3.12 + FastAPI + SQLAlchemy + PostgreSQL
- **Admin Frontend**: Vue 3 + Vite + TypeScript + Tailwind CSS + shadcn-vue
- **User Portal**: Vue 3 + Vite + TypeScript + Three.js
- **LLM Proxy**: Go
- **Deployment**: Kubernetes + Docker
